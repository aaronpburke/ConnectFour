# IO.Swagger - ASP.NET Core 2.0 Server

At-home interview implementation of \"98Point6 Drop-Token\" homework assignment

## Run

Linux/OS X:

```
sh build.sh
```

Windows:

```
build.bat
```

## Run in Docker

```
cd src/IO.Swagger
docker build -t io.swagger .
docker run -p 5000:5000 io.swagger
```
